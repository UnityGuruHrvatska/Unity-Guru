Audio source

Audio source je jedna od Unityjevih zvučnih komponenti koja se koristi za puštanje zvuka.

Kod nje možemo podešavati glasnoću, pitch...

U sljedećim primjerima ćemo vam pokazati kako možete upravljati njenim svojstvima unutar C#-a.


Play i Stop

U sljedećem primjeru ćemo vam pokazati kako moežete pokrenuti i zaustaviti Audio source.

Kada zaustavite određeni Audio source prilikom ponovnog pokretanja on će se pokrenuti od početka, a ne s djela gdje je stao.

Primjer:


public AudioSource zvuk1;

void Start()
{
    zvuk1.Play(); //pokrećemo zvuk1 pri pokretanju igre
}

Primjer:


public AudioSource zvuk1;

void Start()
{
    zvuk1.Stop(); //zaustavljamo zvuk1 pri pokretanju igre
}


Promjena svojstva

U sljedećm primjeru ćemo vam pokazati kako možete mjenjati neka od svojstva.

Primjer:


public AudioSource zvuk1;

void Start()
{
    zvuk1.volume = 1; //stavljamo jačinu zvuka na 1
    zvuk1.pitch = 0.5f; //stavljamo visinu tona na 0.5
    zvuk1.loop = false; //zbog ovoga se zvuk1 nece automatski pokrenuti kada završi
    zvuk1.spatialBlend = 1; //ovdje stavljamo spatialBlend na 1 što znači da će imati 3d efekt(preciznije ćemo ćuti od kud dolazi zvuk)
}
