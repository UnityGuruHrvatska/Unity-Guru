Unos podataka

U C#-u se za korisnikov unos podataka koristi Console.ReadLine().

Primjer:


//ispiše: "Upiši ime:"
Console.WriteLine("Upiši ime:");
                
//stvori string varijablu i u nju spremi podatke koje unese korisnik
string ime = Console.ReadLine();
                
//ispiše vrijednost varijable (ime), što će ispisati korisnikov unos
Console.WriteLine("Ime je: " + ime);


Unos podataka i brojevi

Pošto Console.ReadLine() vraća podatke u obliku stringa, za unos brojeva moramo koristiti pretvaranje vrsta podataka.

Primjer:


//ispiše: "Upiši broj godine:"
Console.WriteLine("Upiši broj godine:");
                
//stvori int varijablu i u nju spremi podatke koje unese korisnik
int godina = Convert.ToInt32(Console.ReadLine());
                
//ispiše vrijednost varijable (godine), što će ispisati korisnikov unos
Console.WriteLine("Broj godina: " + godina);

Ako se u unos upiše krivi format (Npr. tekst kao unos za brojeve) dobit će se iznimka/pogreška (Npr. System.FormatException: 'Input string was not in a correct format.')
