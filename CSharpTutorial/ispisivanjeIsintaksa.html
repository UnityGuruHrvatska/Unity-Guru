C# Sintaksa i ispisivanje

U ovoj lekciji proći ćemo sintaksu C# programskog jezika i pokazati kako ispisivati kod.

Kod pokrećete tako što pritisnete F5 ako koristite Visual Studio Community.

Primjer:


using System;

namespace Tutorial
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello world");
            Console.ReadKey();
        }
    }
}


Objašnjenje sintakse

using - koristimo kada želimo koristiti klase iz određenog imenskog prostora. U ovom primjeru napisali smo using System čime koristimo klase iz imenskog prostora System.

namespace - spremnik za klase i druge imenske prostore. Naš projekt se zove "Tutorial" te iz tog razloga piše namespace Tutorial.

{} - označavaju početak i kraj.

class - spremnik za podatke i metode koji pruža funkcionalnost vašem programu. Svaka linija koda koja se izvršava u C# mora biti unutar klase. Kasnije unutar ove cijeline naučićete više o klasama.

Console.ReadKey() - čeka dok korisnik ne pritisne bilo koju tipku na tipkovnici, a zatim vraća objekt koji predstavlja pritisnutu tipku. Možete ga koristiti kako vam se program ne bi odmah zatvorio te da tako vidite što je ispisano.


Console.WriteLine()

Console.WriteLine() - koristimo ga kada želimo nešto ispisati, a da se nakon toga sljedeći ispis nalazi u novom redu.

Primjer:


using System;

namespace Tutorial
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello world");
            Console.WriteLine("Hello world");
            Console.WriteLine("Hello world");
        }
    }
}


Console.Write()

Console.Write() - koristimo ga kada želimo nešto ispisati, a da se nakon toga sljedeći ispis nalazi u istom redu.

Primjer:


using System;

namespace Tutorial
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.Write("Hello world");
            Console.Write("Hello world");
            Console.Write("Hello world");
        }
    }
}
