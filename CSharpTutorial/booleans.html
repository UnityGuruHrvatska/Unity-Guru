Booleans

Booleans je varijabla koja može poprimiti samo dvije vrijednosti, a to su Istina ili Neistina.

Primjer:


bool foraTutorial = true;
bool narancaJeFina = false;

Console.WriteLine(foraTutorial);   // Ispisuje True
Console.WriteLine(narancaJeFina);   // Ispisuje False


Booleov izraz

Booleov izraz vraća booleovu vrijednost usporedbom vrijednosti/varijabli.

Primjer:


int x = 2;
int y = 5;

Console.WriteLine(x > y); //Ispisuje False
Console.WriteLine(x < y); //Ispisuje True

Primjer:


string lik1 = "Marko";
string lik2 = "Dino";

Console.WriteLine(lik1 == lik2); //Ispisuje False
Console.WriteLine(lik1 != lik2); //Ispisuje True

Primjer:


string lik1 = "Marko";
string lik2 = "Dino";

Console.WriteLine(lik1.Length < lik2.Length); //Ispisuje False
Console.WriteLine(lik1.Length > lik2.Length); //Ispisuje True
